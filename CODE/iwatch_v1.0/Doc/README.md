页面调度的编程思路：

1-初始化的时候，把每个页面.c文件里的，Setup函数，Loop函数，Exit函数的指针保存到一个数组中，这个数组的索引是页面号 ；

2-每次切换页面的时候，调用当前页面的Exit函数，然后调用下一个页面的Setup函数，完成页面的切换，之后再循环执行相应页面的Loop函数  ； 

3-PageRegister_##name()这个函数定义在每个页面.c文件里面最下面，要在PAGE.c文件里面调用，必须先声明，故这里有extern修饰。随后调用每个页面.c文件对应的Page_##name()函数
